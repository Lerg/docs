## Overview

[String](https://docs.coronalabs.com/api/type/String.html) description of an error when [event.isError](/plugin/nfc/event/init/isError.md) is `true`, `nil` otherwise.

Possible values:

- `'This device does not have NFC support.'`
- `'NFC is disabled.'`
