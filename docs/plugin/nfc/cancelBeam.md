# nfc.cancelBeam()

> --------------------- ------------------------------------------------------------------------------------------
> __Type__              [Function](https://docs.coronalabs.com/api/type/Function.html)

> __Return value__      none

> __See also__          [nfc.*](/plugin/nfc/index.md)
> --------------------- ------------------------------------------------------------------------------------------

## Overview

Cancels any previously prepared with [nfc.beam()](/plugin/nfc/beam.md) message.

## Syntax
```lua
nfc.cancelBeam()
```
