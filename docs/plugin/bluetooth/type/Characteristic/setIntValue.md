## Syntax

##### value ~^(required)^~
_[Number][api.type.Number]._ Integer.

##### format ~^(optional)^~
_[String][api.type.String]._ `"sint8"`, `"sint16"`, `"sint32"` (default), `"uint8"`, `"uint16"` or `"uint32"`.

##### offset ~^(optional)^~
_[Number][api.type.Number]._ Integer, default is 0.
